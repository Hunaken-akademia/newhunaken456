# v121 UI / data status fix v2

## 今回直した箇所
1. 開催場一覧を本番配信用の `dist` まで含めて4列化
2. 4列表示に合わせてカード内の文字・余白を調整
3. 展開リハーサルのラベル幅を固定し、全グラフの開始位置を統一
4. 画面上部のデータ状態を、固定の「取得済」から実データ判定へ変更
5. 各項目の「取得待ち」を「未取得」に統一

## 「取得済」になる条件
現在選択中の場・レースについて、次の条件をすべて満たした場合だけです。

- 場が選択済み
- レース番号が選択済み
- 6艇分の選手情報が取得済み
- 展示補正を使う場は、6艇分の展示・1周・回り足が取得済み
- 展示補正を使わない場は、6艇分の選手情報が取得済み

場を選んでいない状態、展示公開前、取得途中、取得失敗時は「未取得」です。
