# HUNAKEN V123 グレード直読みパッチ版

## 区分
パッチ版です。

## 対象
- api/yoso.js のみ

## 目的
これまでの grade / is_ladies 判定は、大会名やページ本文の文字から推測していたため、関係ない文言を拾って PG1 / SG / レディース扱いになる可能性がありました。

V123では、基本方針を以下に変更します。

- 大会名辞書から推測しない
- ページ全体 text.slice(0, 5000) から判定しない
- 表示されている SG / PG1 / G1 / G2 / G3 の文字だけを grade に採用
- 表示されている Ladies / レディース / 女子 の文字だけを is_ladies に採用
- グレード文字が見つからない場合は grade=null に逃がす
- 全員女子選手の判定が取れる場合は、従来どおり is_ladies=true にする

## 触っていないもの
- paid_users
- Stripe
- Googleログイン
- 認証ゲート
- 販売・課金まわり
- race_results 既存データ
- pre_race_status 既存データ

## 注意
このパッチは今後の取得・再取得時の誤判定を防ぐためのものです。
既に races に入っている誤った grade / is_ladies は、このパッチを反映した後に再取得されることで修正されます。
