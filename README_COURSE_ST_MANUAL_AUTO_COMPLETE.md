# レース手動選択統合＋コース別平均ST対応 完全版

## 追加内容
- レースプルダウンを開催中でも常時選択可能に統合
- 公式締切時刻ベースの次R自動選択修正を維持
- ホーム画面の次R表示も公式締切ベースに連動
- 平均STを「現在の進入コース別平均ST」で自動反映
- コース別の母数が少ない選手は全コース共通平均STに自動フォールバック
- 進入変更時は変更後のコース平均STで再計算

## 平均STの表示
- `平均ST（1C）` のように現在進入コースを表示
- コース別母数が足りない場合は `平均ST（全C）` と表示
- 下に `1C平均 n=xx` または `全C平均 n=xx（1C n=不足）` を表示

## デプロイ
GitHubに全ファイルを上書き後、Vercelで Use existing Build Cache を外して Redeploy。
