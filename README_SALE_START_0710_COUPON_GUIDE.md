# 販売開始日 2026/7/10 変更 + 古参クーポン確認版

## 変更内容

- 販売開始日を `2026/7/13 00:00` から `2026/7/10 00:00` に変更
- 販売終了日は変更なし：`2026/8/13 23:59`
- 使用期限は変更なし：`2026/12/31 23:59`
- 1人1回購入・再購入不可は維持
- Stripe Checkout の `allow_promotion_codes: true` は維持

## 古参クーポンについて

コード側は Stripe Checkout でクーポン入力欄が出る設定です。
ただし、クーポン本体はStripe管理画面で作成が必要です。

推奨設定：

- クーポン名：古参100名限定 1000円OFF
- 割引：1,000円OFF
- クーポンコード：KOSAN2026
- 利用上限：100回
- 対象商品：舟券アカデミア予想ツール 2026年版
