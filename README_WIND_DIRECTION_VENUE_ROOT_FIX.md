# 風向き場別補正 根本修正

## 修正内容

鳴門で追い風/向かい風が逆になる問題に対応しました。

原因は、公式beforeinfoの水面気象の風向き画像番号を全場共通の対応表で読んでいたことです。場によってスタート方向・画面上の矢印表現との対応が違うため、同じ画像番号でも追い風/向かい風の意味が逆になるケースがあります。

## 対策

- 文字で「向かい風」「追い風」「左横風」「右横風」が取れる場合は文字を最優先
- 画像番号しか取れない場合だけ、場別補正表を使用
- 鳴門は追い風/向かい風、左右横風を反転補正
- 今後ほかの場で同じズレが出た場合は `WIND_IMAGE_DIRECTION_MAP_BY_VENUE` に追加するだけで修正可能
- 古い風キャッシュを読まないように yoso キャッシュを v114 に更新
