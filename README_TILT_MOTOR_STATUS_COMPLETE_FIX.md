# チルト誤変換・モーター3連対・取得済表示 修正版

ベース: hunaken_paid_auto_wind_seasonal_base_COMPLETE_FIX.zip

## 修正内容

- BOATCAST直前情報のパースで、体重・調整体重・チルトを分離
  - weight: 体重
  - adjust_weight: 調整体重
  - tilt: 本当のチルトのみ
- 調整体重 1.0 などをチルトとして拾わないよう修正
- チルト評価は本当のチルトだけで判定
- 表示文を「✓ コピペ不要：自動取得＋DB成績で反映」から「✓ 取得済」に簡略化
- モーター評価を3連対率重視に変更
  - 6艇中3連対率上位2機はチェック
  - 3連対率60%以上もチェック
  - 3連対率がない場合のみ2連対率で補完
- 風評価は自動取得版の上位機能として継続

## 公開中コピペ版との差について

自動取得版は、風評価・DB補正・季節別場平均1着率・コース別平均ST・進入変更後コース評価・DB選手成績・逃げシミュレーション・決まり手DBなどが入るため、公開中のコピペ版と点数が完全一致しないのは正常です。

今回の修正で、意図しないズレになりやすかった「調整体重がチルトに入る問題」と「モーター3連対率のチェック条件」を修正しています。
