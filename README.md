# v108 桐生 半周ラップ対応版

## 変更点

- 桐生など、BOATCASTのオリジナル展示で「一周」ではなく「半周ラップ」と表示される場に対応。
- `bc_oriten` の見出しが「半周ラップ / まわり足 / 直線」の場合、半周ラップを内部の1周枠へ代替反映。
- 展示タイム・半周ラップ・まわり足・直線の生数値は、従来通り画面上には表示せず内部計算のみで使用。
- 手動貼り付け解析でも「半周ラップ」「半周」を読めるように補強。
- キャッシュ名をv108へ更新し、旧バージョンの展示不足判定を引きずらないように変更。

## 注意

- 江戸川はオリジナル展示なしのため、従来通り展示補正を使いません。
- AI評価に反映する風情報は、従来通り風向き・風速のみです。
