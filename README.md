# v109 風補正二重反映 修正版

## 変更点

- v108 の桐生「半周ラップ」対応と女子選手登録番号リスト完全版を引き継ぎ。
- 「基本場平均1着率」および「進入反映後の基本1着率」から風補正を削除。
- 場平均1着率は、季節別・場別の基本データとして表示する形に整理。
- 「風反映済」の表示を削除。
- 風補正はAI評価内の「風」項目でのみ反映し、枠基準側との二重計算を防止。
- `final1` は「基本場平均＋展示タイム補正」までに変更し、風補正を混ぜないよう修正。
- キャッシュ名と `appVersion` を v109 へ更新。

## 注意

- 江戸川はオリジナル展示なしのため、従来通り展示補正を使いません。
- AI評価に反映する風情報は、従来通り風向き・風速のみです。
- 展示タイム・半周ラップ・まわり足・直線の生数値は、画面上には表示せず内部計算のみで使用します。
