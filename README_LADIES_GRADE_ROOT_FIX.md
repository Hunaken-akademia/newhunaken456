# 女子戦・グレード判定 根本修正

## 修正内容

- 「女子戦」フィルターはグレードとは独立して `is_ladies=true` を参照します。
- レディースオールスターをG2、レディースチャンピオン/クイーンズクライマックス等をPG1、オールレディースをG3として優先判定します。
- ヤングダービーを「ダービー」の語だけでSGに誤分類しないよう、固有大会名を先に判定します。
- ヴィーナスシリーズ、女子リーグ、女子王座、賞金女王、スピードクイーンメモリアル等の旧称・表記揺れを追加しました。
- 男女W優勝戦、レディースVSルーキーズ等は大会全体を女子戦にせず、6艇全員が女子選手のレースだけを女子戦にします。
- 過去データも `race_results` の登録番号から6艇全員女子を判定するSQLを追加しました。

## 実行SQL

Supabase SQL Editorで `sql/backfill_race_grades_and_ladies_now.sql` を実行してください。
