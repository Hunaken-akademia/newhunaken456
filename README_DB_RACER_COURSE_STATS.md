# DB枠別成績 自動反映

## 追加内容

自動取得で6艇の登録番号が入った時点で、Supabase の `race_results` から選手別・進入コース別成績を自動取得します。

- 今期（直近365日扱い）
- 直近6ヶ月
- 直近3ヶ月
- 直近1ヶ月
- 当地（直近365日・現在の開催場）

を自動計算し、画面の「選手成績の区分」プルダウンで切り替えできます。

## これで不要になるもの

枠別成績のコピペは基本不要です。
補助データ一括欄は、平均ST・決まり手・逃げシミュレーションの貼り付け補助用として残しています。

## Supabase権限確認

もし画面に `DB選手成績：失敗` が出る場合は、SQL Editorで以下を実行してください。

```sql
grant usage on schema public to anon, authenticated;
grant select on public.race_results to anon, authenticated;
```

## 表示

艇カードの下部に以下のように表示されます。

`枠別DB（直近6ヶ月 n=12） 1着 25.0% 2連 50.0% 3連 66.7%`

nはその選手がその進入コースで走った母数です。
